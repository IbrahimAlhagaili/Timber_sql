Product Sales Report Script
Instructions

1. Ensure SQL*Plus is installed in your directory.
2. Open a terminal in the directory where SQL*Plus is installed.
3. Use the `START "path"` command, replacing "path" with the full path to the provided script file.
   Example: `START "C:\path\to\script.sql"`
4. The script will generate a report showing the quantity and value of sales for product categories based on category, customer, supplier, and total.
5. The report includes information such as category name, customer name, supplier name, quantity ordered, value ordered, and product names.
6. The output will be saved to the "C:\ProductSales.txt" file.
7. Review the generated report for insights into product sales.

Note: Make sure to replace the example path with the actual path to the script file on your PC.
