Show All suppliers and products supplied Script

Instructions:

1. Ensure SQL*Plus is installed in your directory and you have run the build and populate files.
2. Open a terminal in the directory where SQL*Plus is installed.
3. Use the `START "path"` command, replacing "path" with the full path to the provided script file.
   Example: `START "D:\cprg250s\script.sql"`
4. All the supplier and product names should be displayed along with the IDS. 

Note: Make sure to replace the example path with the actual path to the script file on your PC.
