STEP 1: In order to create the tables with constraints
    1.Locate the "Build Folder".
    2.Inside the Build Folder, you will find a script and spool file for creating the tables.
    3 Run the create table script by copying the path and using the start command in SQL Plus. (With adequate permissions)

STEP 2: To populate the tables
    1. Find and open the "Populate" folder
    2. Inside the Populate folder, you will find a script and spool file used for populating the tables.
    3. Run the populate script by copying the path and using the start command in SQL Plus. This script will insert data into the tables.
